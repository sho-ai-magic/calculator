# **🎀 Kawaii さんすうれんしゅう（ガチャ＆シール帳版）**

お子様が楽しみながら算数を学べる、ゆめかわいいデザインのフラッシュカード・アプリケーションです。  
ビルド不要の静的ファイルのみで動作するため、サーバー不要でブラウザさえあればどこでも遊べます。

## **✨ 特徴**

* **ゆめかわいいUI**: 女の子が喜ぶパステルカラーのピンク、水色、黄緑を基調とした、キラキラで可愛いデザイン。  
* **4つの学習モード**: たしざん、ひきざん、九九（かけざん）、わりざんに対応。  
* **ゲーミフィケーション要素**:  
  * **レベル＆XP**: 正解すると経験値（XP）が貯まり、レベルアップ！  
  * **シールガチャ**: パーフェクト賞やレベルアップで手に入る「ガチャチケット」で、可愛いシール（全50種類）を集めよう！  
  * **スタンプカード**（開発予定）: 毎日遊ぶとスタンプが貯まり、ボーナスチケットがもらえる予定の機能です。※現在は未実装です。  
* **スマホ最適化**: スマートフォンの一画面にぴったり収まるレスポンシブ設計。入力は専用のテンキーで行うため、キーボード操作は不要です。  
* **データ保存**: 学習の記録や集めたシールは、お使いのブラウザの中（LocalStorage）に自動保存されます。保存に使うキー名は `kawaii_arithmetic_save_v7_lucky` です。データはインターネット上ではなく、そのブラウザの中だけに保存されます（別の端末やブラウザには引き継がれません）。

## **🎮 あそびかた**

1. **設定を選ぶ**: 「だん」を選ぶか「シャッフル」かを選んで、スタート！  
2. **もんだいに答える**: 画面のテンキーをタップして答えを入力。「OK」を押して正誤判定。  
3. **ごほうびをゲット**: 全問正解（パーフェクト）するとガチャチケットがもらえます。  
4. **ガチャを回す**: 「がんばり記録」画面からチケットを使ってガチャに挑戦！新しいシールをコレクションしましょう。

## **🛠 技術スタック**

* **Frontend**: HTML5, CSS3 (Tailwind CSS)  
* **Logic**: Vanilla JavaScript  
* **Fonts**: M PLUS Rounded 1c (Google Fonts)  
* **Persistence**: Window.localStorage

## **📁 ファイル構成**

役割ごとにファイルを分けています。「〇〇を変えたい」ときにどのファイルを見ればよいかの対応表もあわせてご覧ください。

```
calculator/
├── index.html          … 画面のマークアップ（見える部品の配置）
├── css/
│   └── style.css       … 見た目（色・デザイン・アニメーション）
└── js/
    ├── config.js       … 設定データ（シール一覧・称号・セーブ名・初期値）と共有変数
    ├── sound.js        … 効果音（ボタン音・正解音・ガチャ音など）
    ├── level.js        … レベルと経験値（XP）の計算
    ├── storage.js      … セーブデータの読み込み・保存
    ├── ui.js           … 画面の切り替えと表示の更新
    ├── rewards.js      … ごほうび（チケット・XP・紙吹雪・ビンゴ・ガチャ）
    ├── quiz.js         … 問題づくり・問題表示・答え合わせ・結果集計
    └── main.js         … アプリの起動とボタンの動作の結びつけ
```

### どのファイルが何を担当するか（変えたいこと別）

| やりたいこと | 見るファイル |
| --- | --- |
| 音を変えたい | `js/sound.js` |
| 見た目（色・デザイン）を変えたい | `css/style.css` |
| 画面の文字や配置を変えたい | `index.html` |
| 問題の出し方を変えたい | `js/quiz.js` |
| ごほうび（チケット・シール・ガチャ）を変えたい | `js/rewards.js` |
| レベルや必要XPの計算を変えたい | `js/level.js` |
| 保存のしかた（セーブ）を変えたい | `js/storage.js` |
| シール一覧・称号・初期値などのデータを変えたい | `js/config.js` |
| 画面の表示（上部バー・記録画面）を変えたい | `js/ui.js` |
| ボタンを押したときの動きを変えたい | `js/main.js` |

## **📌 技術メモ**

* **ビルド不要・静的ファイルのみ**: ビルドツールは使いません。ファイルをそのままブラウザで開くだけで動きます。
* **ES modules（import/export）は使わない**: ダブルクリックでファイルを開いたとき（`file://`）に動かなくなるため、通常の `<script src>` タグで順番に読み込んでいます。
* **scriptタグの読み込み順は変えないこと**: `index.html` の下部にある読み込み順（config → sound → level → storage → ui → rewards → quiz → main）は、あとのファイルが前のファイルの内容を使う「依存関係」になっています。順番を入れかえると動かなくなります。

## **🚀 つかいかた**

2. ブラウザ（Chrome, Safari, Edgeなど）で `index.html` を開きます。  
3. すぐに練習を開始できます！
